# meteor-nsq

## Usage

    import sharp from "meteor/c9s:nsq";

## Testing Package

    meteor test-packages .

## Publishing Package

    meteor publish

    meteor admin get-machine os.linux.x86_64
    meteor publish-for-arch c9s:sharp@0.16.2_2

    meteor admin get-machine os.osx.x86_64
    meteor publish-for-arch c9s:sharp@0.16.2_2
